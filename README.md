# 8-bit Microcontroller using VHDL

## Project Overview
This project implements an 8-bit Microcontroller using VHDL and simulates the design in Vivado 2024.1.

## Objective
To design and verify an 8-bit microcontroller architecture and demonstrate instruction execution using simulation.

## Tools Used
- Verilog
- Vivado 2024.1
- FPGA Design Flow

## Features
- Arithmetic operations
- Register operations
- Control unit
- Instruction execution
- Simulation verification

## Project Structure
src/ → VHDL source files  
tb/ → Testbench files  
constraints/ → XDC files  
results/ → Simulation outputs  

## How to Run
1. Open Vivado
2. Open project (.xpr)
3. Run Synthesis
4. Run Simulation
5. Observe waveforms

## Results
Simulation completed successfully and outputs verified.
